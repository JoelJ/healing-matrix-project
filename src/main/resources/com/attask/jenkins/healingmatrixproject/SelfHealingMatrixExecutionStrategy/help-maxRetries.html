The maximum number of times any qualifying build should be retried.

If 0, the build will run exactly 1 time (retried 0 times).
If 1, the build will run at most 2 times (1 time initially, plus potentially 1 more time).
If 2, the build will run at most 3 times (1 time initially, plus potentially 2 more times).
etc.

The number of times a build has been retried is available as a parameter named SELF_HEAL_RETRY_COUNT. This value is between 0 and the number provided in this build, inclusive.
